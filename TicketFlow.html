🔗 Publicar en Microsoft Teams vía SharePoint
Sigue estos 6 pasos para dejar TicketFlow disponible como pestaña en tu canal de Teams.
1
Descarga el archivo TicketFlow.html — es este mismo archivo, con todo incluido (sin dependencias externas).
2
Ve a tu sitio de SharePoint → Documentos → crea una carpeta llamada Apps → subcarpeta ticketflow.
3
Sube el archivo TicketFlow.html dentro de esa carpeta. Haz clic derecho → Copiar vínculo → asegúrate de que sea accesible para tu organización.
4
Crea o edita una página de SharePoint → agrega un Web Part de tipo Embed → pega la URL del archivo HTML.
5
Publica la página de SharePoint y copia su URL completa (algo como https://tuempresa.sharepoint.com/sites/tuSitio/SitePages/TicketFlow.aspx).
6
En Teams abre tu canal → + Agregar pestaña → elige Sitio web → pega la URL de la página de SharePoint → ponle el nombre Solicitudes → Guardar.
⚠️ Nota sobre iframe en SharePoint

Si el Web Part Embed no renderiza el HTML directamente, usa en su lugar el Web Part File viewer apuntando al archivo .html, o considera alojar el archivo en Azure Static Web Apps (gratis) para tener una URL HTTPS limpia que Teams acepte sin restricciones.

📋 Qué pedirle a TI si hay bloqueos

1. Permiso para subir archivos HTML a bibliotecas de SharePoint
2. Permiso para agregar Web Parts de tipo Embed en páginas
3. Confirmar que la URL de SharePoint no está bloqueada para iframes en Teams
